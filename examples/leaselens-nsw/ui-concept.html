LL-05-AskList

Ask before signing

  • Bond: Can you confirm the bond lodgement method and timing?
  • Repairs: Can existing defects be added to the condition report before move-in?
  • Special conditions: Which clauses add obligations beyond the standard agreement?
  • Human review: If an answer changes money, access, or termination rights, seek tenancy advice.
