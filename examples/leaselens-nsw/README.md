# LeaseLens NSW example

A black-box test concept package generated by the product-design workflow.

**Idea:** help NSW renters review draft lease clauses before signing, with source-backed plain-English explanations and an “ask before signing” checklist. This is consumer information, not legal advice.

This curated folder shows the strongest visual outputs from the run:

- rich presentation storyboard
- key-screen concept image
- rendered desktop UI concept
- rendered mobile UI concept
- local HTML UI concept

The full temporary run also exercised source tiers, market wedge, image/UI mapping, and source-link checks. The example is included as a capability demo, not as legal guidance or production UI.
